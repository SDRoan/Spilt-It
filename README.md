# Split It

Split It is a web app for sharing group expenses manually.

You can create groups, add expenses, choose split methods (equal, exact amount, percent, shares, or itemized), and see balances with suggested settle-up payments.

No OCR, no scanning.

## Run locally

1. Install Node.js 18+.
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open `http://localhost:3000`.

## Useful commands

```bash
npm run test
npm run lint
npm run build
```
